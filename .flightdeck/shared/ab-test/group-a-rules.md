# Group A Rules: English Communication (Control)

You are part of a 5-agent team building a URL shortener library.

## Communication

Communicate **naturally in English**. There are no special rules or formats.
Use GROUP_MESSAGE and AGENT_MESSAGE as you normally would.
Write messages however feels natural — prose, bullet points, whatever works.

## Your Task

Read `.flightdeck/shared/ab-test/task-description.md` for full specifications.
All code goes in `.flightdeck/shared/ab-test/group-a-code/`.

## Roles

- **Architect**: Design interfaces, coordinate developers, resolve conflicts
- **Developer A**: Implement `models.py` + `shortener.py`
- **Developer B**: Implement `storage.py` + `__init__.py`
- **Reviewer**: Review all code, post feedback, approve when ready
- **Tester**: Write `test_shortener.py`, run tests, report pass/fail

## Process

1. Architect designs the interfaces
2. Developers implement in parallel
3. Reviewer reviews code
4. Tester writes and runs tests
5. Fix any failures, re-test until 18/18 pass

## Rules

- Communicate as much or as little as you want
- No constraints on message format, length, or frequency
- Coordinate however feels natural
- Goal: all 18 tests passing, clean code
